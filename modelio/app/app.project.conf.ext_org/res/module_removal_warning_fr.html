Confirmation du retrait du module

Vous êtes sur le point de retirer un module du projet.

Cette opération implique les conséquences suivantes:

  • Toutes les annotations (stéréotypes, tagged values, notes, etc.) apportées par ce module vont être supprimées.
  • Les fonctionnalités et commandes apportées par ce module ne seront plus disponibles.
Attention : cette opération est irréversible, réinstaller le module ne restaurera pas les annotations.

N.B. Des éléments de modèle en lecture seule peuvent utiliser des annotations (stéréotypes, tagged values, notes...) apportés par le module.
Les librairies de modèle en lecture seule doivent donc être retirés du projet avant de pouvoir retirer le module.
Pour cela utilisez la commande 'Configuration\Bibliothèques...'.
