model

This metamodel conforms to the specification of the ArchiMate Enterprise Architecture modeling language, a visual language with a set of default iconography for describing, analyzing, and communicating many concerns of Enterprise Architectures as they change over time.

The standard provides a set of entities and relationships with their corresponding iconography for the representation of Architecture Descriptions.
