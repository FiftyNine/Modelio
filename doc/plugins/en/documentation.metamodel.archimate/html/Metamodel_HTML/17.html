PassiveStructureElement
Passive structure elements are the objects that can be accessed by behavior elements.

A passive structure element is a structural element that cannot perform behavior. Active structure elements can perform behavior on passive structure elements.

In the domain of information-intensive organizations, which is the main focus of the language, passive structure elements are usually information or data objects, but they may also be used to represent physical objects.
