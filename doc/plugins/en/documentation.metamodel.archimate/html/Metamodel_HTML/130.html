passive
Technology layer passive structure elements

A technology object models the passive structure elements that are used and processed by the infrastructure.

An artifact is a physical piece of information that is used or produced in a software development process, or by deployment and operation of a system. It is the representation, in the form of, for example, a file, of a data object, or an application component, and can be deployed on a node. The artifact element has been taken from UML.
