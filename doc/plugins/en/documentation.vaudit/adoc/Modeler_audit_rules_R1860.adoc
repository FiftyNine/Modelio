===== Summary

In an _Interface_, the visibility of all _Features_ must be public.

[[Details]]

[[details]]
===== Details

An _Interface_ is by definition a public contract.

All its features must be public.

[[Tips]]

[[tips]]
===== Tips

Change the visibility of all features to public.
