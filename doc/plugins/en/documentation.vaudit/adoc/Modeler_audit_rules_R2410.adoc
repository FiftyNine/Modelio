===== Summary

An _Operation_ cannot own both 'create' and 'destroy' stereotypes.

[[Details]]

[[details]]
===== Details

Constructor and destructor are exclusive stereotypes: a method cannot be both a constructor and a destructor.

[[tips]]
===== Tips
