===== Résumé

Tous les paramètres d'une opération doivent avoir un nom différent.

===== Détails

Les paramètres sont identifiés de façon unique grâce à leur nom.

[[Conseils]]

[[conseils]]
===== Conseils

Renommez les paramètres en conflit.
