===== Résumé

Il ne peut y avoir de cycles dans le graphe des _relations d'extension_.

===== Détails

Un cycle de _relation d'extension_ est interdit.

[[conseils]]
===== Conseils

Supprimez une des _relations d'extension_ du cycle.
