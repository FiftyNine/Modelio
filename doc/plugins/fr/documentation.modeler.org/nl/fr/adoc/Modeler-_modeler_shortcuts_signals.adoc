= Raccourcis de démasquage sur les signaux

|Constraints |SHIFT + N
|Notes |SHIFT + N
|Element imports |SHIFT + U
|Package imports |SHIFT + U
|Parent generalizations |SHIFT + G
|Child generalizations |g
|Attributes |SHIFT + A
|Operations |SHIFT + O
|Parts |SHIFT + I
|Ports |SHIFT + P
|Associations |SHIFT + L
|Dependencies |SHIFT + D
|Nodes |CTRL + O
|=================================
